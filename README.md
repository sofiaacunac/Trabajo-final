# Trabajo-final
Este proyecto analiza el impacto del precio sobre las ventas en el sector moda. Se estiman modelos econométricos para medir la elasticidad de la demanda, validar su robustez con pruebas estadísticas y proyectar ventas a 6 meses con intervalos de confianza. Además, se desarrolla un dashboard interactivo para apoyar decisiones de pricing.
